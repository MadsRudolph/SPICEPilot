========================================
HOW TO SEE THE GRAPH IN KICAD SIMULATOR
========================================

The simulation RAN successfully (you saw it in the log).
Now you need to PLOT the results!

========================================
STEP-BY-STEP TO GET THE GRAPH:
========================================

In the KiCad Simulator window:

1. RIGHT-CLICK in the empty plot area
   (the big blank space where graph should be)

2. Select "Add Signals..." from menu
   (or click the "Add Signals" button in toolbar)

3. A dialog will open showing available signals

4. Look for and SELECT:
   - v(vout)
   - OR V(vout)
   - (capitalization might vary)

5. Click OK

6. The signal should appear in the plot!

7. To see it as dB (Bode plot):
   - In the left panel, you'll see the signal listed
   - RIGHT-CLICK on "V(vout)" in the signals list
   - Select "Show as" ‚Üí "Magnitude (dB)"

8. For phase:
   - Add V(vout) again
   - Right-click ‚Üí "Show as" ‚Üí "Phase"

========================================
IF YOU DON'T SEE ANY SIGNALS:
========================================

Option A: Check what's available
---------------------------------
1. Look in the SPICE Console/Log tab
2. Scroll through the output
3. Look for node names like:
   - vout
   - n_d2
   - n_d3
   - n_tail

Option B: Use specific node
---------------------------------
Try adding these signals one by one:
- V(vout)
- v(vout)
- vout
- V(n_d3)

Option C: Check the log output
---------------------------------
1. In Simulator window, click "SPICE Console" tab
2. Look for lines that say "No. of Data Rows"
3. Below that, you should see node names
4. Try adding those exact names
